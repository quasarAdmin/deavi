HELP

How to start an algorithm

  • To start an algorithm click on the Algorithms option in the side menu and select Start

  • You can always check the status of the execution by going to the Pipeline page. Here you can also see the results of such execution once it is finished.

How to query the mission archives

  • To query the Gaia archive click on the Queries option in the side menu and select Gaia Here you will find all the options to select a region of the sky and retrieve the data from that region.

  • To query the Herschel science archive click on the Queries option in the side menu and select Herschel Here you will find all the options to select a region of the sky and retrieve the data from that region.

  • You can always check the status of the query by going to the Status page. Here you can also see the results of such query once it is finished.

How to manage your Data

  • To manage your files go to Data on the side menu and select Data
