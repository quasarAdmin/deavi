Algorithms

{% csrf_token %}

This section contains the set of scientific algorithms developed in the context of the Star Form Mapper project. Each provided algorithm within DEAVI has been developed with one scientific goal in mind. A few simple examples have been included for the purpose of showing some of the functionalities implemented in DEAVI. The complete list of algorithms can be found below by navigating through the different tabs more.

Each algorithm needs a given set of input parameters and a data set over which to run. Explanations are provided inside the corresponding section of each algorithm.

DEAVI also contains Tutorials in the Guide section that guide you through each one of the algorithms with an example. Scientific publications explain the details behind each algorithm and should be read for the correct interpretation of the results. These scientific references are also provided in the algorithms corresponding sections.
